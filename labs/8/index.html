Lab 8 - JavaScript DOM manipulation and event handling

Duration: 2 hours. This lab introduces JavaScript DOM manipulation and event handling to create interactive web pages.

In this lab, you will build a simple chatbot that displays questions and answers from a predefined question bank. The chatbot will use DOM manipulation to update the page content dynamically and event handling to respond to user interactions. You will learn how to select elements, modify their content, and attach event listeners to create an interactive experience. By the end of the lab, your chatbot should be able to cycle through questions and display corresponding answers based on user input.

On this page: Overview · Theory · Lab Setup · Tasks · Push, publish, and submit · Extra · References.

Overview

Objectives: Master DOM manipulation techniques including selecting elements, modifying content and attributes, and dynamically creating elements. Learn to handle various events such as clicks, key presses, and form submissions using event listeners.

Concepts: DOM tree, selectors, element manipulation, event listeners, and event propagation.

Tasks: Select and modify DOM elements, add event listeners, and create interactive features.

Deliverable: Upload to GitHub Classroom an interactive page with DOM manipulation and event handling implemented.

Theory

DOM Basics

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects; that way, programming languages can interact with the page.

A web page is a document that can be either displayed in the browser window or as the HTML source. In both cases, it is the same document but the Document Object Model (DOM) representation allows it to be manipulated. As an object-oriented representation of the web page, it can be modified with a scripting language such as JavaScript.

Introduction to JavaScript

JavaScript, often abbreviated as JS, is a versatile and essential programming language for web development.

JavaScript is a scripting language primarily used for adding interactivity and dynamic behavior to websites. It allows web developers to create features such as form validation, animations, interactive maps, and much more. JavaScript is executed in the user's web browser, which means it can respond to user actions without the need to reload the entire web page.

To link JavaScript to HTML, you can use the <script> element in your HTML file's <head> or <body> section. You can include JavaScript code directly within the HTML file or link to an external JavaScript file.

Here's a simple example of a JavaScript code snippet to display an alert when a button is clicked:

<!DOCTYPE html>
<html lang="en">
<head>
    <title>JavaScript Example</title>
</head>
<body>
    <button onclick="showAlert()">Click me</button>

    <script>
        function showAlert() {
            alert("Hello, JavaScript!");
        }
    </script>

</body>
</html>

In this code, we create a function showAlert that displays a pop-up alert with the message "Hello, JavaScript!" when the button is clicked.

Here's an example of linking an external JavaScript file:

<!DOCTYPE html>
<html lang="en">
<head>
  <script src="myscript.js"></script>
</head>
<body>
    <!-- Your HTML content here -->
</body>
</html>

In this example, the src attribute of the <script> tag points to an external JavaScript file named myscript.js. This separates your JavaScript code from your HTML, making it easier to manage and reuse.

Variables

Variables are used to store information. To create a variable in JavaScript, use the let keyword. The statement below creates (in other words: declares) a variable with the name message:

let message;
message = 'Hello'; // store the string 'Hello' in the variable named message
message = 'World!'; // value changed

We can declare multiple variables like:

let user = 'John', age = 25, message = 'Hello';

let user = 'John';
let age = 25;
let message = 'Hello';

let user = 'John',
     age = 25,
     message = 'Hello';

JavaScript supports different data types: strings, numbers, booleans, null, undefined, etc.

let str = "Hello";
let str2 = 'Single quotes are ok too';
let phrase = `can embed another ${str}`;

let nameFieldChecked = true;
let ageFieldChecked = false;

let age_1 = null;
let age_2; // undefined

let x = 1;
x = -x; // -1

let x = 1, y = 3;
alert( y - x ); // 2

let fruits = ["Apple", "Orange", "Plum"];
console.log( fruits[0] ); // Apple

Objects

The last important concept needed for our website is the JavaScript object. Objects are simply containers for key-value pairs, similar to what you'd find in a dictionary.

let fullName = {
  firstName: 'John',
  lastName: 'Doe'
}

console.log(fullName.firstName);

Control Structures: if, while, do..while, for

Control structures allow you to control the flow of your program based on conditions or loops.

IF statement:

let accessAllowed;
let age = prompt('How old are you?', '');
if (age > 18) {
  accessAllowed = true;
} else {
  accessAllowed = false;
}
console.log(accessAllowed);

WHILE loop:

let i = 0;
while (i < 3) { // shows 0, then 1, then 2
  console.log( i );
  i++;
}

DO…WHILE loop:

let i = 0;
do {
  console.log( i );
  i++;
} while (i < 3);

FOR loop:

for (let i = 0; i < 3; i++) { // shows 0, then 1, then 2
  console.log(i);
}

Functions

Functions are the main "building blocks" of the program. They allow the code to be called many times without repetition.

Function definition:

function name(parameter1, parameter2, ... parameterN) {
 // body
}

Example:

function showMessage() {
  console.log( 'Hello everyone!' );
}

showMessage();
showMessage();
