Lab 2: HTML Forms and Accessibility

This page corresponds to Lab 2 on the Assignments page. Use the objectives, concepts, tasks, and deliverable described there.

Objectives

  • Build accessible forms

Concepts

<form>, <label>, input types, ARIA basics.

Tasks

  1. Create a contact form
  2. Use proper labels (e.g. for / id)
  3. Add required fields and pattern validation where appropriate
  4. Use semantic grouping with <fieldset> where it helps

Deliverable

Form page committed and validated. Submit via the course platform.
